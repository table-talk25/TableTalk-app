# üåç Guida alla Pluralizzazione Dinamica - TableTalk

## üìñ Panoramica

TableTalk ora supporta la **pluralizzazione dinamica** per tutte le lingue supportate, utilizzando le regole native di i18next. Questo sistema garantisce che i testi si adattino correttamente al numero di elementi, seguendo le regole grammaticali di ogni lingua.

## üéØ Problemi Risolti

### **Prima (Sistema Statico)**
- ‚ùå Testi hardcoded per singolare/plurale
- ‚ùå Traduzioni non naturali ("1 TableTalk¬Æ" vs "2 TableTalk¬Æ")
- ‚ùå Supporto limitato per lingue con regole complesse (es. arabo)
- ‚ùå Manutenzione difficile per nuove lingue

### **Dopo (Sistema Dinamico)**
- ‚úÖ **Pluralizzazione Automatica**: i18next sceglie la forma corretta
- ‚úÖ **Lingue Complesse**: Supporto per arabo (5 forme), francese (3 forme)
- ‚úÖ **Manutenzione Semplice**: Aggiunta nuove lingue senza modifiche al codice
- ‚úÖ **UX Naturale**: Testi che suonano naturali in ogni lingua

## üèóÔ∏è Architettura

### **Configurazione i18n**
```javascript
// src/i18n.js
pluralRules: {
  // Italiano: 1, 2+, 0
  it: {
    numbers: [1, 2, 0],
    plurals: function(n) {
      if (n === 1) return 0;        // "1 TableTalk¬Æ"
      if (n >= 2 && n <= 19) return 1; // "2 TableTalk¬Æ"
      return 2;                     // "0 TableTalk¬Æ"
    }
  },
  // Inglese: 1, other
  en: {
    numbers: [1, 2],
    plurals: function(n) {
      return n === 1 ? 0 : 1;      // "1 TableTalk¬Æ" vs "2 TableTalk¬Æ"
    }
  }
}
```

## üîß Configurazione Avanzata

### **Regole Personalizzate**
```javascript
// Per lingue con regole complesse
pluralRules: {
  custom: {
    numbers: [0, 1, 2, 3, 4, 5],
    plurals: function(n) {
      if (n === 0) return 0;
      if (n === 1) return 1;
      if (n === 2) return 2;
      if (n >= 3 && n <= 10) return 3;
      if (n >= 11 && n <= 20) return 4;
      return 5;
    }
  }
}
```

### **Separatori Personalizzati**
```javascript
i18n.init({
  pluralSeparator: '_',    // mealCount_one
  keySeparator: '.',       // meals.mealCount
  contextSeparator: '|'    // mealCount|context
});
```
